Cross-chain Interoperability

Darwinia Network provides an entrance to the Polkadot ecology for projects that have been deployed on public blockchains such as Ethereum and BSC.

Smart Contract Solution

Darwinia Smart App is compatible with the Ethereum virtual machine, and supports DApps such as Defi and NFT, which can be easily migrated to Polkadot.

Faster and Easier User Experience

Darwinia network can bring users a very high-quality experience, including very low transaction fees and ultra-fast transaction confirmation speed.

User-driven Network Governance

Darwinia Network has a sound governance system where all token holders can submit proposals. Upgrades to the network are coordinated on-chain and enacted autonomously, ensuring that Darwinia’s development reflects the values of the community and avoids stagnation.

Powering the Darwinia Network

The native tokens for Darwinia Network are RING, KTON.

KTON is a derivative commitment token of RING, which encourages long-term involvement.

RING

The native tokens for Darwinia Network is RING, RING can be used as gas for transactions. Gas include transaction fees, contract execution fees, network bandwidth charges, storage fees, and more.

KTON

To encourage users to make long term commitments and pledge, users can choose to lock RING for 3 - 36 months in the process of Staking, and the system will offer a KTON token as reward for users participating in Staking.
